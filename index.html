Synlighet etterpå

Tidsplan

  1. Start

    Kickoff + tema slippes

    Kort intro, regler, og dere kan finne team.

  2. Underveis

    Bygg, test og iterer

    Små mål. Fokus på en spillbar prototype.

  3. Frist

    Send inn

    Last opp lenke og kort beskrivelse før nedtellingen er ferdig.

  4. Etterpå

    Visning + utmerkelser

    Spill hverandres prosjekter og del favoritter.

Regler

1) Team og deltakelse

  • Team: 1–4 personer.
  • Du kan delta alene.
  • Vær hyggelig og inkluderende.

2) Hva skal sendes inn?

  • En spillbar prototype.
  • Lenke til itch.io eller GitHub (anbefalt).
  • Kort beskrivelse + controls.

3) Innhold og rettigheter

  • Bruk egne eller lovlige assets (CC/royalty-free).
  • Krediter musikk/lyd/bilder du bruker.
  • Innhold må være trygt og passende for arrangementet.
